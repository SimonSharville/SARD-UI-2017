Table Lists

This sections shows status alerts as well as tool tips and popovers

 

When creating a table that contains Status Lights and Link Icons you will need to add certain classes.

To add Status Icons within a table, you should add .status-col to every <th> and <td> within that column. This will make that column 5% the table width and remove the excess padding from those cells. The Status Lights themselves should be placed within a <div> tag, within the relative <td>. i.e: <td class="status-col"> <div class="status status-alert"></div> </td> To include Link Icons before the text link, Add the class .link-icon to the table i.e <table class="table table-hover link-icon‚Äù>. This will place the small arrow icon before every text link within the table.

Also to hang the text correctly when the link text is longer than 1 line, wrap the link in a <p> tag. AND remove any tags between them.

Status Alerts

This sections shows status alerts as well as tool tips and popovers.

These are all within a Panel, BUT NOT within a Table.

 

Status Alerts

Status alerts are designed to show the status of a document in a list by colour coding.
They follow the same principles of the Bootstrap alerts but also have their status initial within them.
Success = Green
Warning = Yellow
Danger - Red

We have also added
Alert = Orange
None = Clear

Tooltips

Popovers are added by using a link with NO url, like this:
<a class="pull-right glyphicon glyphicon-info-sign" data-toggle="popover" title="Add popover title here" data-placement="bottom" data-content="Add popover content here."></a>

Tooltips are activated by hovering over them while Popovers are activated by clicking on them.
To avoid visual clutter all are set to float right of the list item they are within.

Table Lists

This sections shows status alerts as well as tool tips and popovers

 

When creating a table that contains Status Lights and Link Icons you will need to add certain classes.

To add Status Icons within a table, you should add .status-col to every <th> and <td> within that column. This will make that column 5% the table width and remove the excess padding from those cells. The Status Lights themselves should be placed within a <div> tag, within the relative <td>. i.e: <td class="status-col"> <div class="status status-alert"></div> </td> To include Link Icons before the text link, Add the class .link-icon to the table i.e <table class="table table-hover link-icon”>. This will place the small arrow icon before every text link within the table.

Also to hang the text correctly when the link text is longer than 1 line, wrap the link in a <p> tag. AND remove any tags between them.

          </td>
          <td>
            <p><a href="#">Appraisal Jul 2016</a></p>
          </td>
        </tr>
        <tr>
          <td class="status-col">
            <div class="status status-danger"></div>
          </td>

      </tbody>
    </table>
  </div><<!-- END Panel body -->
</div><!-- END Panel -->

 

Note: to make the table responsive the table needs to be wrapped with a .table-responsive class.

Rows & Perforations

Everything is contained within ROWs <div class="row"></div>. If you wish to add a perfration between 2 ROWs then simply add a new ROW between them with a class of perf-grey <div class="row perf"></div>.

The perforation is made of repeating dot images of the same colour as the body background colour. If you wish to have a different background colour than we will need to make new images of that colour and use them.

 

If you wish to have a coloured row inbetween 2 rows then do the following:
After the first row add a new row with an added class of .perf, followed by a second row with two added classes of .row-[color] and .row-between-perfs.
This row is written:
<div class="row perf"></div>
<div class="row row-grey row-between-perfs"></div>

This row is written:
<div class="row perf"></div>
<div class="row row-turq row-between-perfs"></div>

Responsive Tables

To make the table responsive the table needs to be wrapped with a .table-responsive class.
In most cases that class should be added to the containing panel. i.e:
<div class="panel panel-default table-responsive">.

When creating a list like above, the list sit’s within a panel. The panel body contains a series of DIVs, one for each list item, in this case with a class of notice.
To add the Status Alert, simply add the following classes to any div: status status-STATE. i.e.
<div class="notice status status-success">

Table Status Alerts

When creating a table that contains Status Lights and Link Icons you will need to add certain classes.

To add Status Icons within a table, you should add .status-col to every <th> and <td> within that column. This will make that column 5% the table width and remove the excess padding from those cells. The Status Lights themselves should be placed within a <div> tag, within the relative <td>. i.e: <td class="status-col"> <div class="status status-alert"></div> </td> To include Link Icons before the text link, Add the class .link-icon to the table i.e <table class="table table-hover link-icon”>. This will place the small arrow icon before every text link within the table.

Also to hang the text correctly when the link text is longer than 1 line, wrap the link in a <p> tag. AND remove any tags between them.

Here is an example of a complete responsive table within a panel:
